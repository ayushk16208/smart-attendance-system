> A web-based Smart Attendance System for colleges and universities that automates attendance tracking, eliminates manual entry, and provides real-time reports.

🔗 **Live Demo:** [smart-attendance-byhk.onrender.com](https://smart-attendance-byhk.onrender.com)

---

## 📸 Preview

| Feature | Description |
|---|---|
| 👨‍🎓 Student Portal | Students can view their attendance records |
| 👨‍🏫 Teacher Portal | Teachers can mark and manage attendance |
| 📊 Reports | Real-time attendance reports and analytics |
| 🔐 Login System | Secure login for students and teachers |

---

## ✨ Features

- ✅ Automated attendance marking
- ✅ Teacher and Student login system
- ✅ Real-time attendance reports
- ✅ Class-wise attendance tracking
- ✅ Reduce manual errors in attendance
- ✅ Accessible from any device via browser
- ✅ Clean and simple UI
